数据库设置

数据库文件在settings.py文件中，将对应name等进行更改

cnpm下载

在文件 /frontend> cmd下执行
cnpm -v #查看状态（在此之前，于官网下载nodejs，将npm镜像淘宝地址配置成功）
cnpm install #下载

一共42个包，在第13个时会卡一会，不慌。成功后，有个√
关于错误：
①不是内部或外部命令
1）nodejs环境变量配置错误
2）命令输错了（注意，是cnpm）
3）Data数据包中有数据，不能为空

数据库连接

1）任务管理器打开—》服务—》MySQL57—》右键运行
2）开始，MySQL，连接，输密码

3）本机Navicat连接
连接后先建库，然后表右键运行.sql文件
导入.sql文件时报错，438数据成功400个
不勾选第二个，成功

项目跑起来

npm run dev

②在项目cmd中输入
